# Automation Anywhere - Task 2 Playwright Automation

## Setup Instructions
1. Run `npm install`
2. Run `npx playwright install`
3. Make sure `.env` is populated with credentials:
   ```
   CONTROL_ROOM_URL=...
   AA_USERNAME=...
   AA_PASSWORD=...
   ```
4. Run `npx playwright test` to execute the automation.
5. Run `npx playwright test --headed` to execute the automation with browser UI.
